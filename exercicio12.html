Formas de Utilizar o CSS:

Existem três formas principais de utilizar CSS em um projeto web: inline, interno (ou incorporado) e externo.

CSS Inline:

Aplica estilos diretamente a elementos HTML individuais,

utilizando o atributo style dentro da tag de abertura do elemento.

Desvantagem: Não é recomendado para projetos maiores, pois torna o

código HTML mais poluído e dificulta a manutenção.

CSS Interno (ou Incorporado):

Define estilos dentro da seção (head) do documento HTML, utilizando a tag (style)
